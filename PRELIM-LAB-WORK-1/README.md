BubbleSort Java Program
Description

This is a Java console application that reads integers from a text file, allows the user to choose a sorting algorithm and order (ascending or descending), sorts the data, and displays the sorted result along with the time taken to sort.

Supported sorting algorithms:

Bubble Sort

Insertion Sort

Merge Sort

Requirements

Java JDK 8 or later

A text/dataset file containing integers (separated by spaces or new lines)
