Description

BubbleSortGUI is a Java Swing-based desktop application that loads data from a text file and sorts it using different sorting algorithms. Users can choose the algorithm and sorting order (ascending or descending), then view the sorted output and the time taken to perform the sort.

Supported sorting algorithms:

Bubble Sort

Insertion Sort

Quick Sort

Merge Sort

The program sorts text lines (strings) alphabetically.

Requirements

Java JDK 8 or later

Any operating system that supports Java Swing

A text file containing data (one value per line)

How to Run
1. Save the File

Save the source code as:

BubbleSortGUI.java

2. Compile the Program

Open a terminal or command prompt in the folder containing the file and run:

javac BubbleSortGUI.java

3. Run the Program

After compiling successfully, run:

java BubbleSortGUI


The graphical interface will open automatically.

How to Use the Application

Click Load File and select a text file.

The file contents will appear in the Original Data panel.

Select a sorting algorithm from the dropdown menu.

Choose Ascending or Descending order.

Click Sort.

View:

Sorted output in the Sorted Data panel

Execution time in the Time Taken panel
